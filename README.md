Liste des commandes

/pvp-create : Créer un événement.
Syntax :
/pvp-create name:value mode:value date:value duration:value ( duration : optionnel )

/pvp-edit : Editer un événement.
Syntax :
/pvp-edit id:1 name:RBG Full Tryhard
/pvp-edit id:1 mode:3v3 duration:90
/pvp-edit id:1 date:2026-03-21 21:00

/pvp-list : Lister les événements avec leurs IDs.
Syntax :
/pvp-list
/pvp-list mode:value

/pvp-show : Afficher les événements.
Syntax :
/pvp-show
/pvp-show id:value

/pvp-delete : Supprime un événement.
Syntax :
/pvp-delete id:value

/pvp-clean-expired : Supprime les événements expirés manuellement.
Syntax :
/pvp-clean-expired
